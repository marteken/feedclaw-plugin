# Connect FeedClaw

FeedClaw uses standard OAuth — Claude handles the connection automatically.

If you see a "Connect FeedClaw" button, click it and log in with your FeedClaw account.

If tools are not available yet:
1. Add the connector manually in Claude Settings → Connectors → Add connector → `https://www.feedclaw.io/api/mcp`
2. Claude will then prompt you to authorize

Once connected, FeedClaw tools work seamlessly in every session.
