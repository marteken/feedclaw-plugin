# X (Twitter) Publishing Guidelines

## Character limit

- Hard limit: **280 weighted chars**
- Emoji and non-ASCII Unicode (✅ 🔥 etc.) count as **2 chars**, not 1
- Count before calling create_draft — the API will reject if over 280
- If over limit: rewrite shorter, do not truncate mid-sentence

## Format

- Plain text only — no markdown, no asterisks, no bullet points
- Short paragraphs, single line break between thoughts
- No hashtags unless the brand explicitly uses them

## Copy principles

- Strong hook on the first line: bold claim, surprising stat, or direct question
- Never start with "I" or "We launched" — lead with value, not the company
- Vary hooks across a batch: insight, story, hot take, data point, question
- Never repeat the same hook style twice in a batch

## Common mistakes to avoid

- Starting every tweet with the brand name
- Using the same text as the email newsletter
- Ending with weak CTAs like "Check it out!" — be specific
